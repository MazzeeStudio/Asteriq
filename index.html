What is Asteriq?

Running a dual-stick or full HOTAS setup in Star Citizen usually means having vJoy, HidHide, and JoystickGremlin open alongside the game. Asteriq replaces all three with a single application.

Input is handled via SDL2, the same foundation JoystickGremlin uses, which tends to be more reliable than DirectInput for this kind of work. Device-to-vJoy assignments are tracked by hardware identity rather than port order, so the configuration stays consistent regardless of how things are plugged in.

Features

Device Management

Each physical controller gets assigned to a specific vJoy slot — that slot is what Star Citizen sees. Assignments are tied to the device's hardware identity, so nothing shifts if you replug cables in a different order.

If HidHide is installed, Asteriq can hide your physical devices from other applications whilst forwarding. This stops SC from seeing both the real hardware and the virtual devices at the same time.

Input Mapping

Axes support linear, S-curve, exponential, and custom response curves. In custom mode you place control points manually and get a live input vs output readout as you move the stick.

You can merge two physical axes onto a single virtual output, which is useful if rudder input is split between a twist grip and separate pedals. Buttons have four modes: normal, toggle, pulse, and hold.

Star Citizen Bindings

The Keybindings tab reads your current SC profile and shows every action alongside its bound input. You can export from there to actionmaps.xml. Asteriq reads your SC config but never writes to it, so existing bindings are not at risk.

Profiles and Themes

Save different device and mapping setups as named profiles and switch between them from within the app. The interface has a handful of manufacturer-themed colour schemes (Drake, Aegis, Anvil, Origin and others). There is also a built-in update checker that handles the download and exe swap in one click.

Getting Started

  1. 1
    Download and run Asteriq

    Grab Asteriq.exe from the button above and run it. No install needed, no separate runtime — the exe is self-contained.

  2. 2
    Install drivers if you need them

    Virtual joystick forwarding requires vJoy 2.x. If you also want SC to only see the virtual devices rather than your physical hardware as well, install HidHide on top of that. Asteriq runs without either, but most people will want at least vJoy.

  3. 3
    Assign your devices

    Open the Devices tab and assign each physical controller to a vJoy slot. That slot number is what Star Citizen will see as a joystick.

  4. 4
    Set up your mappings

    In the Mappings tab, dial in your axis curves and set button modes as needed.

  5. 5
    Start forwarding, then launch the game

    Click Start Forwarding and launch Star Citizen. Asteriq runs in the background and routes input through the virtual devices.
