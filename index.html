What is Asteriq?

Asteriq is a single application for managing HOTAS controllers in Star Citizen. It replaces the separate tools you would typically need — a vJoy configurator, HidHide, and JoystickGremlin — by bringing device assignment, input mapping, and Star Citizen binding export together in one place.

It uses SDL2 for input, which is the same approach used by JoystickGremlin and is more reliable for this purpose than DirectInput. Physical devices are assigned to specific vJoy virtual joystick slots by you, explicitly, and those assignments persist across sessions so your configuration stays stable.

Features

Device Management

Assign your physical HOTAS devices to specific vJoy virtual joystick slots. Asteriq maintains stable device identity across sessions, so your configuration does not shift when controllers are connected in a different order. Online and offline status is shown in real time.

HidHide integration hides your physical devices from other applications whilst Asteriq forwards their input through the virtual devices, preventing Star Citizen from seeing duplicate joysticks.

Input Mapping

Map axes, buttons, and hats from any physical device to virtual outputs. Axes support four response curve modes — linear, S-curve, exponential, and fully custom with draggable control points — with a live input/output preview as you adjust.

Multiple physical inputs can be merged onto a single virtual axis, useful for combining a twist grip and a separate rudder pedal. Buttons support normal, toggle, pulse, and hold modes.

Star Citizen Bindings

Browse your Star Citizen keybindings directly in Asteriq and export them to actionmaps.xml format. Asteriq never modifies your existing Star Citizen configuration — it is export only, so your bindings remain entirely under your control.

Actions are grouped by category and display the current binding for each input, which is useful for confirming your vJoy slot assignments match what Star Citizen expects.

Profiles and Themes

Save and switch between multiple configuration profiles. Asteriq uses a dark, futuristic interface with several manufacturer-inspired colour themes — Drake, Aegis, Anvil, Origin, and others — that change the accent colour throughout the application.

Automatic update checking notifies you when a new release is available. Updates can be downloaded and applied with a single click.

Getting Started

  1. 1
    Install the prerequisites

    You will need the .NET 8 Runtime, vJoy 2.x, and HidHide installed before running Asteriq.

  2. 2
    Download and run Asteriq

    Grab the latest Asteriq.exe from the Releases page and run it. No installation required.

  3. 3
    Assign your devices

    Go to the Devices tab and assign each physical controller to a vJoy slot. This determines how Star Citizen sees your devices.

  4. 4
    Configure mappings and start forwarding

    Set up axis curves and button modes in the Mappings tab, then click Start Forwarding to begin routing input through the virtual devices.
