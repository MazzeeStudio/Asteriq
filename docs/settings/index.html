Settings

The Settings tab covers configuration management, system driver status, network forwarding setup, visual themes, and general application preferences.

Configuration Management​

A configuration stores all your device assignments, axis curves, and button mappings. You can have multiple configurations and switch between them — useful if you fly different ship types with different mapping setups.

ActionDescription
NEWCreates a new empty configuration and makes it active
DUPLICATECopies the active configuration to a new one
IMPORTLoads a configuration from an exported file
EXPORTSaves the active configuration to a file for backup or sharing
DELETERemoves the configuration permanently

Click the configuration name to rename it.

System​

The System panel shows driver status and general application settings.

Driver status​

  • vJoy — shows whether the vJoy driver is installed and how many virtual devices are configured
  • HidHide — shows whether HidHide is installed

Click Driver Setup to open the driver setup dialog, where you can install or reinstall either driver.

General settings​

SettingDescription
Auto-load last configurationAutomatically activates the last used configuration on startup
Close to trayMinimises to the system tray instead of closing when you click the window close button
Client Only ModeHides the Devices and Mappings tabs. Intended for RX machines receiving input over the network. See Client Only Mode
Check for updates automaticallyChecks GitHub for a newer release on startup and prompts if one is available

Network​

The Network section configures forwarding between machines. See the Network Forwarding page for full setup instructions.

Network toggle — enables or disables network forwarding support. When disabled, Asteriq operates in local-only mode.

Role — select TX (transmitter, has physical controllers) or RX (receiver, running Star Citizen).

Visual Themes​

Asteriq includes several colour themes. The core themes are:

CodeNameCharacter
MIDMidnightDark blue — the default
MTXMatrixGreen terminal
AMBAmberWarm orange CRT
ICEIceCold cyan

Manufacturer themes include Drake, Aegis, Anvil, Argo, Crusader, Origin, MISC, and RSI — each styled to match that manufacturer's visual identity in-game.

Background atmosphere controls (grid intensity, glow, noise, scanlines, vignette) let you tune the visual density independently of the colour theme.

Support​

The Support panel contains links to support the project and join Star Citizen using a referral code.

  • Buy Me a Coffee — opens the project's support page in your browser
  • Star Citizen Referral — copy the referral code or open the Star Citizen signup link directly
