Getting Started

This page walks you through setting up Asteriq from scratch — from installation to your first forwarding session.

1. Download and install​

Download Asteriq.zip from the latest release and extract the contents to a permanent folder — for example C:\Program Files\Asteriq\. Asteriq must stay in that folder; it is not an installer and should not be run from inside the zip or from your Downloads folder.

Run Asteriq.exe. Windows may show a SmartScreen warning because the application is not code-signed. Click More info, then Run anyway.

tip

Keep all files in the extracted folder together. Asteriq requires vJoyInterface.dll, Asteriq.dll.config, and the Images/ folder to be alongside the executable.

2. Driver setup​

On first launch, Asteriq checks for the drivers it needs and shows the Driver Setup screen.

vJoy Virtual Joystick is required. It creates virtual joystick devices that Star Citizen sees as your controllers. Click Install in the vJoy panel to download and run the installer.

HidHide Device Hiding is recommended. It hides your physical devices from other applications so that only the virtual devices are visible to Star Citizen. Click Install in the HidHide panel to install it, then configure device visibility within HidHide directly — Asteriq does not manage this.

If the automatic install does not work, use the Manual Download links at the bottom of the dialog to download each driver from GitHub.

When both drivers show as installed, the status at the top of the dialog reads All drivers installed and ready. Click Continue. Tick Don't show again if you do not want this screen to appear on future launches.

3. Create a configuration​

Asteriq stores your device assignments, axis curves, and button mappings in a configuration. On first launch there is no active configuration, so the first thing to do is create one.

Open the Settings tab and find the Configuration Management panel.

Click NEW, enter a name for your configuration, and confirm. The new configuration becomes active immediately.

4. Assign your devices​

Open the Devices tab. Your connected physical controllers appear in the left-hand list under the Physical category.

Select a physical device by clicking it. The right-hand panel shows the device's details and available actions.

Click MAP 1:1 TO VJOY. A dialog appears listing your existing vJoy devices alongside their current axis, button, and POV counts. Devices marked [OK] have enough axes and buttons to accommodate the physical device you selected. Choose one and click MAP 1:1 to confirm.

If none of your existing vJoy devices are suitable, click + Configure new vJoy device... to create one configured to match the physical device's requirements, then select it and confirm.

warning

If the MAP 1:1 TO VJOY button is not visible, vJoy is either not installed or no vJoy devices are configured. Check driver status in the Settings tab.

Repeat for each controller you want to forward.

5. Start forwarding​

With your devices assigned, click Start Forwarding in the Devices tab.

Asteriq is now routing input from your physical devices through the vJoy virtual devices. You can minimise the window — Asteriq continues running in the system tray.

6. Launch Star Citizen​

Start Star Citizen. SC will detect the vJoy virtual devices as joysticks. Because the virtual devices are always present in a consistent order, your bindings will remain stable regardless of how or when you plugged in the physical hardware.

tip

If you are setting up SC bindings for the first time, head to the SC Bindings tab in Asteriq to assign actions and export your control profile.

Stopping​

Click Stop Forwarding in the Devices tab, or right-click the Asteriq icon in the system tray and select Stop.
