Asteriq

Asteriq is a HOTAS management application for Star Citizen. It brings device assignment, axis curve configuration, Star Citizen bindings, and network input forwarding into one window.

How it works​

Star Citizen identifies joysticks by their position in the device list — Joystick 1, Joystick 2, and so on. Physical USB devices are unreliable in this regard: replug them in a different order and their numbering shifts, breaking in-game bindings.

Asteriq solves this with a three-layer model:

Physical device  →  vJoy virtual device  →  Star Citizen
  (your HOTAS)       (stable, numbered)       (sees JS1, JS2…)

Your physical controller is assigned to a specific vJoy slot. Asteriq reads from the physical device and forwards its input to the corresponding virtual device. Star Citizen only ever sees the virtual devices, which are always present in the same numbered positions regardless of how or when you plugged in the physical hardware.

What you can do​

FeatureDescription
Device ManagementAssign physical devices to vJoy slots, set joystick order for SC, manage vJoy virtual devices
MappingsConfigure axis response curves, deadzone, button modes, and merge multiple axes
SC BindingsLoad a Star Citizen control profile, assign inputs across your devices, detect conflicts, and export
Network ForwardingForward your HOTAS input over a local network to another PC
SettingsManage configurations, visual themes, system driver status, and updates

Requirements​

  • Windows 10 or 11 (64-bit)
  • vJoy 2.x — required for virtual device forwarding
  • HidHide — optional; if you want to hide physical devices from other applications, install and configure it separately
note

Asteriq displays HidHide driver status but does not manage device visibility itself. Configure which applications can see your physical devices within HidHide directly.
